Website accessibility and known bungalow access notes for guests planning a stay.

Website accessibility

This website is built with semantic HTML, visible text links, alt text for key imagery, keyboard-accessible navigation, responsive layouts and no autoplay media.

Known bungalow access notes

The public listing says the bedroom is on the ground floor. It also notes a step up into the shower. This may not be suitable for every guest, so confirm your needs before booking.

Contact for access questions

Email bookings@thebungalowburnley.co.uk before booking if you have accessibility questions.

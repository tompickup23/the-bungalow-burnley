# The Bungalow Burnley — Claude Code Context

## Overview
Property website for The Bungalow in Burnley.

**Stack**: HTML | CSS | JavaScript
**Hosting**: GitHub Pages
**Branch**: main

## Key Patterns
- Static property showcase site
- Image galleries and property details
- Contact information and location maps
- Mobile-first responsive design

## Rules
- British English throughout
- High-quality property images (optimised for web)
- Clear contact details and CTAs
- Mobile-first — most property searches are on mobile
- SEO: property-relevant keywords in titles and descriptions
- Fast loading — compress images, lazy-load below fold
